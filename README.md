# Suraksha Sakhi

## Team: Dukhi Devs

Suraksha Sakhi is a web-based legal awareness platform focused on **women and child protection laws in India**.
The goal is to make complex legal information easier to understand for non-lawyers, without replacing legal professionals.

This repository contains the **frontend** and supporting **microservices** for the platform.

## Features

- Simplified explanations of women & child safety laws
- News and updates on relevant legal cases and bills
- Petition and signature collection for important issues
- Myth-busting around common legal misconceptions
- Case references and real-world examples

## Project Structure

```
/
├── frontend/        # Web frontend
├── news/            # News microservice
└── README.md
```

## Tech Stack

- Frontend: Next.js / React
- Backend Services: FastAPI (Python)
- Database: PocketBase
- AI / NLP: RAG-based pipelines with safety checks

## Disclaimer

This platform is for **legal awareness only**.
It does **not** provide legal advice and should not be treated as a replacement for professional legal consultation.
